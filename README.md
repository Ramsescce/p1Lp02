Vetor alocado dinâmicamente<br>
Considere um vetor alocado dinamicamente dentro de uma função, em que cada elemento seja um<br>
valor não inteiro. Nesse vetor, devem ser armazenados os valores diferentes de zero contidos<br>
em vetor de 100 posições passando como parâmetro da função. Assim, a função deve alocar<br>
dinamicamente um vetor e preenche-lo com os valores diferentes de zero do vetor passado como<br>
parâmetro e retornar o endereço para esse vetor alocado dinamicamente. <br>
float * novo_vetor(float v[]);
